# Tradex

Modern Point of Sale for Retail Businesses

A fast, intuitive POS system built for shops and small businesses. Manage sales, track inventory, and operate with confidence.

---

## Features

- Fast checkout with minimal friction  
- Real-time inventory management  
- Multiple payment methods: cash, transfer, card, mobile money  
- Daily, weekly, and monthly sales reporting  
- Professional receipt generation  
- Role-based user permissions  
- Percentage and fixed-amount discount engine  

---

## Tech Stack

- Laravel 11  
- Livewire 3  
- Tailwind CSS  
- MySQL  
- Alpine.js  

---

## Build Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Dabeey/tradex.git


## 2. Install dependencies:

composer install  
npm install  


## 3. Configure environment:

Copy .env.example to .env  
Update database credentials  


## 4. Generate application key:

php artisan key:generate  


## 5. Run migrations:

php artisan migrate  


## 6. Start development server:

php artisan serve  
npm run dev
